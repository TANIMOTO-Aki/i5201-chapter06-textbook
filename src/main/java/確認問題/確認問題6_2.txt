配列に関する次の説明のうち正しいものには○、誤っているものには×を付けなさい。
①[○]    int[] score = new int[10]; により要素10個のint型配列を確保する。
②[×]    int[] score = new int[10]; 配列の添字の最大値は10である。
③[×]    score[5] = 10; は、配列scoreの先頭から数えて5個目の要素に10を代入する。
④[○]    int[] score;とint score[];は同じ意味である。
⑤[×]    int score[10];により、配列の宣言と同時に配列を確保する。

②…配列の先頭は0からのため要素数10の場合、添字はscore[0]～[9]
③…②と同じ理由でscore[5]は先頭から6番目になる
⑤…宣言と確保を同時に行う記述は「int[] score = new int[10];」
